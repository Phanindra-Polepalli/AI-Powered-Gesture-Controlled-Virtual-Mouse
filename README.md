# AI-Powered Gesture-Controlled Virtual Mouse

## 📌 Overview
This project allows users to control the computer mouse using hand gestures captured through a webcam.

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## ✨ Features
- Cursor movement using hand tracking
- Click using pinch gesture
- Scroll functionality
- Drag and drop support

## ▶️ How to Run
1. Install required libraries:
   pip install opencv-python mediapipe pyautogui

2. Run the file:
   python main.py
